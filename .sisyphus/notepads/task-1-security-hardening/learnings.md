# 学习记录 - 后端安全加固

- 统一从环境变量读取 JWT 密钥，避免硬编码，生产环境必须配置 JWT_SECRET。
- 提供基于 gin 的 CORS 中间件，灵活读取环境变量 CORS_ALLOWED_ORIGINS 配置，空时允许所有来源。
- 引入基于 IP 的简单内存令牌桶限流，默认 60 req/min，登录接口 10 req/min；实现无外部存储（如 Redis）。
- 将中间件注册引入路由初始化，确保所有请求在进入业务逻辑前经过保护。
- .env.example 增加了 CORS_ALLOWED_ORIGINS 的演示，方便开发环境快速上手。

- 下一步优化点：对限流实现增加更细粒度的控制、异常日志、以及对高并发场景的性能压力测试。
